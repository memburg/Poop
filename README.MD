## What is 💩.js?
💩.js (/puːp ˈdʒeɪ ɛs/) is an attempt to write less code in JavaScript, is a tiny library with a short version of most commonly used functions in JavaScript.

## Constants
All the ```Math.LETTER``` things simplified.
- ```E```
- ```LN2```
- ```LN10```
- ```LOG2E```
- ```PI```
- ```LOG10E```
- ```SQRT1_2```
- ```SQRT2```
